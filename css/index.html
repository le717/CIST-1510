Looking for the CSS?

You are probably here because you want to read the CSS I wrote to create this site. Below, you will find a directory of all the style sheets I used. For informational purposes, I wrote it using Sass 3.3.12 using the SCSS syntax, whose files end with the .scss extension.
The compiled CSS uses the .css extension, while the minified CSS (compiled by Sass) used by the site has the .min.css extension (not listed below).
